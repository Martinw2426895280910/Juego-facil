# 🧟 ZOMBIES: The Last Stand

Juego de survival en el navegador — sin dependencias externas.

## 🎮 Cómo jugar
| Acción | PC | Móvil |
|---|---|---|
| Mover | WASD / Flechas | Joystick virtual |
| Disparar | Espacio / Z | Botón 🩸 |
| Granada | G | Botón ⏱️ |
| Subir nivel | Llegar a 🛸 | Igual |

## ⚡ Features
- Auto-disparo al zombie más cercano (rango 250px)
- Power-ups: Medkit, Shotgun, Machine Gun, Speed, Nuke, Grenade
- Boss 👹 desde nivel 3
- Highscore guardado en localStorage
- Audio Web API (sin librerías)
- 100% responsive — funciona en móvil y PC

## 🚀 Deploy
Publicado en [Vercel](https://vercel.com)

## 👾 Stack
HTML + CSS (Tailwind CDN) + JavaScript vanilla (React via ESM)
